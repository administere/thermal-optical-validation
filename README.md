# 热光混合处理器 — 工程验证

> 用光做计算，用热来控制光。一份诚实的工程分析。

## 这是什么

我在做一个新型光计算芯片的可行性分析。基本想法很简单：

```
一束激光 → 穿过一片特殊的薄膜 → 被探测器接收
              ↑
         薄膜被加热到 242°C
         加热改变了薄膜对光的"折射能力"
         于是光的强弱被精确控制
         这就是"热光计算"
```

关键技巧是**光子复用**：一个光脉冲能穿过 2048 个调制点，相当于做了 2048 次乘法。光在这里既传递信息，又维持温度。

## 六个核心问题

| # | 问题 | 目前结论 | 
|---|------|:--:|
| 1 | 242°C 能维持住吗？ | 实验证实材料可以，但靠 VCSEL 自身加热不够，需要外加热源 |
| 2 | 计算速度够快吗？ | 普通热光方式约 0.5 次/秒写权重，跟 H100 差不多 |
| 3 | 光信号够强吗？ | 够，SNR 32 dB |
| 4 | 电路复杂吗？ | 不复杂，2048 个 ADC 就够了 |
| 5 | 省电吗？ | 如果用量子拍频机制（17.6 GHz），能省 ~200 倍；普通方式只省几倍 |
| 6 | 能造出来吗？ | 需要先做实验验证，第一步约 $5K，2 周 |

## 能耗对比

| 方式 | 每 dot product 能耗 | vs H100 |
|------|:--:|:--:|
| DiSubPc·C70 自加热 @ 850nm | 377,000 fJ | 8× |
| DiSubPc·C70 外加热 @ 570nm | 706,000 fJ | 4× |
| TiO₂ 外加热 @ 570nm | 693,000 fJ | 4× |
| **DiSubPc·C70 量子拍频** | **14,000 fJ** | **199×** |

> H100 GPU 做同样的事需要约 2,900,000 fJ。

**核心结论：普通热光方式只比 GPU 省几倍的电。只有量子拍频机制能实现数量级的突破。材料选 TiO₂ 还是 DiSubPc，差距只有 20%。但用不用量子拍频，差距是 50 倍。**

## 先说清楚：这个分析的边界在哪

我不想假装这个东西比实际情况更靠谱。以下三节请认真看——它们告诉你哪些数字你可以信，哪些不能信，以及整个分析的逻辑地基是什么。

### 数据可信度分级

本项目所有物理参数来自 Nature Photonics 2026 论文的 8 份补充材料（MOESM1-8），但并不是所有数字都同等可靠：

| 等级 | 来源 | 涉及参数 |
|:--:|------|---------|
| 🟢 **实测** | MOESM7 光热曲线、MOESM8 瞬态吸收 | 242°C 可达性（>45min）、激发态寿命 τ=4.2ns、量子拍频 17.6 GHz |
| 🟡 **文献值** | 已发表 SOI 平台数据 | Si 折射率 n=3.476、dn/dT=1.8×10⁻⁴、Pπ=12-25 mW |
| 🟠 **外推** | MOESM6 紫外-可见光谱 → 乌尔巴赫带尾模型 | 850nm 吸收系数 α≈350 cm⁻¹（不确定度 ~10×） |
| 🔴 **估算** | 有机半导体典型值，非 DiSubPc·C70 实测 | dn/dT 值、热导率、热容、242°C 下的光谱数据 |

> **最大不确定性**：① 850nm 吸收系数的外推值可能在 35-3500 cm⁻¹ 之间；② 242°C 下的材料光学性质完全没有实测数据。这两点是整个热光计算路线最大的物理风险。

### 核心假设

以下每一条如果被证伪，结论就需要修正。我把最危险的排在前面：

| # | 假设 | 如果错了会怎样 |
|:--:|------|------|
| 1 | 量子拍频（17.6 GHz）可以耦合到光场并调制透射率 | **整个量子路线不成立。** 论文只证明了拍频产生热，未证明光调制。这是核心风险 |
| 2 | DiSubPc·C70 在 242°C 长期运行（>1000h）不分解 | MOESM7 只测了 45 分钟。如果材料在高温下缓慢分解，这条路需要完全不同的材料 |
| 3 | 850nm 吸收系数的外推值在正确数量级 | 如果实际 α<35 cm⁻¹，自加热方案完全不可行；如果 α>3500 cm⁻¹，吸收足够但不确定度意味着我们其实不知道为什么 |
| 4 | dn/dT 在 242°C 下不退化 | 有机半导体 dn/dT 通常随温度升高而降低。如果退化 10×，热筛效果大打折扣 |
| 5 | 大面积薄膜（6×6cm）厚度均匀性可达 ±5μm | 30μm 有机薄膜的大面积旋涂/蒸镀均匀性是工艺挑战，不是买设备就能解决的 |
| 6 | 探测器阵列的 ADC Walden FOM 可达 15 fJ/conv | 当前商用最先进 ~50 fJ/conv，15 是路线图目标，不是已有产品 |

### 已知局限 — 这个分析回答不了的问题

| 问题 | 为什么回答不了 | 需要什么 |
|------|:--:|------|
| 242°C 下 1000 小时可靠性 | 材料太新，没人做过 | 加速老化实验 |
| 量子拍频 → 光调制的转换效率 | 物理机制未验证 | 泵浦-探测实验 @ 242°C |
| 2048 通道的制造良率 | 没有工艺线和 PDK | 流片或代工厂 PDK 数据 |
| 热光计算 vs 电光计算的实际延迟 | 本分析只做能耗，不做延迟建模 | 系统级时钟周期分析 |
| 与硅光子平台的工艺兼容性 | DiSubPc·C70 从未在 CMOS 线上沉积过 | 工艺开发项目 |

## 这些方法靠不靠谱？行业标准对标

> 结论先说在前面：方法论上全部有文献依据。实现上是分析级精度，不是生产级（没有工艺库，无法 tape-out）。以下逐一说明。

### 方法论评级

| 方法 | 行业标准 | 本项目 | 差距 |
|------|:--:|:--:|------|
| **FDTD 全波电磁仿真** | ✅ 标准 | ✅ MEEP（MIT） | MEEP 是学术界标准 FDTD 求解器。与商业工具 Lumerical FDTD 相比，MEEP 缺少 GUI 和自动化网格优化，但数值精度同级。TMM 解析模型与 FDTD 交叉验证是论文标准做法 |
| **Clements 酉分解** | ✅ 标准 | ✅ 完整实现 | Clements et al. (Optica 2016) 是片上可编程光子网格的**标准架构**，被 MIT、Oxford、Xidian 等所有光计算团队采用。本项目的 N(N-1)/2 MZI 拓扑、逐列消元算法、D 相位修正完全符合论文 |
| **SVD 矩阵乘法** | ✅ 标准 | ✅ 完整实现 | M=UΣV† 是光学矩阵乘法的标准范式（Miller, Photonics Research 2013）。两个 Clements 网格 + 衰减器阵列 = 通用矩阵乘法器。与 Lightmatter、Celestial AI 专利架构同源 |
| **2D 有限差分热求解** | ✅ 方法标准 | ⚠️ 简化版 | 5 点模板 FDM + 稀疏求解是标准方法。但工业级工具（COMSOL、Ansys Icepak）使用 3D 非均匀网格 + 对流/辐射边界条件。本项目只做 2D 稳态 + 叠加近似，等效于 COMSOL 的 1/10 精度 |
| **热串扰矩阵 Cᵢⱼ** | ✅ 标准 | ✅ 完整实现 | 串扰矩阵是光电子设计中的标准表征方法。与 COMSOL 文章中"thermal crosstalk coefficient"定义一致。叠加原理在线性热传导中严格成立 |
| **Pπ 品质因数** | ✅ 标准 | ✅ 完整实现 | Pπ（π 相移所需电功率）是热光移相器的**行业标准 FOM**。Harris et al. (Opt. Express 2014) 实验值 12-25 mW 是本项目的对标基准 |
| **TMM 传输矩阵法** | ✅ 标准 | ✅ 完整实现 | 薄膜光学的标准方法（Macleod, *Thin-Film Optical Filters*）。商业软件 Essential Macleod、TFCalc 使用同样算法。本项目 TMM 与 FDTD 交叉验证是正确做法 |
| **乌尔巴赫带尾外推** | ✅ 标准 | ✅ 合理使用 | 半导体物理中分析亚带隙吸收的标准模型。Urbach (1953) 原始论文已被引 4000+ 次。外推不确定度大是因为缺实测数据，不是方法问题 |
| **噪声预算分解** | ✅ 标准 | ✅ 完整 | 散粒噪声 + 热噪声 + RIN + 暗电流 + 背景光 + TIA 输入噪声 → 6 源分解是光接收机设计标准（Agrawal, *Fiber-Optic Communication Systems*） |
| **SNR → ENOB 换算** | ✅ 标准 | ✅ 完整 | SNR(dB)=6.02×ENOB+1.76 是 ADC 行业标准公式（IEEE Std 1241） |

### 与工业级工具的差距

## 材料分析

以下是为感兴趣的人准备的背景材料。如果你只关心工程结论，上面已经够了。

### DiSubPc·C70

四川大学 2026 年在《Nature Photonics》上发了一篇论文，发现了一种叫 DiSubPc·C70 的有机共晶材料。它被光照后能自己升温到 242°C，而且内部电子会产生每秒 176 亿次的"量子拍频"。

论文只用它来做光热转换（烧水、淡化海水）。我第一个想到用它来做光计算——这个材料在 242°C 下对光的折射能力会随温度变化，可以用来做光学乘法。

**这个材料有什么特别的：**

- **17.6 GHz 量子拍频**：电子在两种状态之间每秒振荡 176 亿次。如果能把这种振荡转化成光信号的强弱变化，计算速度会比普通方式快 400 倍。
- **极性晶体结构**：材料的分子排列不对称，这是产生量子拍频的结构基础。跟它类似的 C60 版本是对称的，拍频只有 6 GHz。

**这个材料有什么问题：**

- **850nm 激光吸收太弱**：材料最适合的吸收波长是 570nm 左右（绿光），850nm 的近红外光吸收只有绿光的 1/230。我们用了数学方法估算 850nm 的吸收率，但不确定度很大。
- **242°C 下的性能没人测过**：所有光谱数据只测到室温（27°C），242°C 时材料会发生什么，全靠推测。
- **量子拍频还没被用来做计算**：论文只证明了拍频产生了热，没证明拍频能调制光信号。

### 替代材料 TiO₂

如果不追求量子拍频，只做"加热改变折射率"这件事，TiO₂（二氧化钛）是更好的选择：

- 折射率随温度变化的幅度是 DiSubPc·C70 的 3 倍
- 完全透明，光不会被浪费
- 熔点 1840°C，242°C 对它来说不算什么
- 标准半导体工艺就能做

但它没有任何量子效应。如果你只做普通热光计算，TiO₂ 更好。如果你想赌量子突破，DiSubPc·C70 是唯一的选项。
